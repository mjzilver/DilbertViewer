# Dilbert Viewer

Dilbert Viewer is a Qt-based desktop application for viewing Dilbert comics and searching them by date, tag, or transcript.

## Features
- Browse Dilbert comics in a clean desktop interface
- Search comics by:
  - Publication date
  - Tags
  - Transcript text
- Local comic storage for offline viewing

## Legal Notice
This application does **not** include any Dilbert comics by default.

Dilbert Viewer includes an optional web scraper that downloads comics from an archived version of the official Dilbert website where these comics were legally distributed. Downloaded comics are intended for **personal use only**.
